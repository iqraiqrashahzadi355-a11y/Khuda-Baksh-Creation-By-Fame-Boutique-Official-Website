## 🌟 About This Project

A **fully functional, multi-page business website** for Khuda Baksh Creation By Fame Boutique — Multan's trusted women's clothing boutique with a **4.8⭐ Google Rating**. Built entirely with HTML, Tailwind CSS, and vanilla JavaScript. No framework, no backend, no database.

> 📍 Tariqabad, Multan, Punjab, Pakistan &nbsp;|&nbsp; 📞 +92 302 6825777 &nbsp;|&nbsp; 🕐 11AM–11PM (Fri: Closed)

<details open>

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| **HTML5** | Semantic structure across all 7 pages |
| **Tailwind CSS (CDN)** | Styling, responsiveness, Material Design 3 colors |
| **Vanilla JavaScript** | Filters, search, counter animations, WhatsApp form logic |
| **Google Fonts** | Playfair Display (headings) + Montserrat (body) |

| **Google Maps Embed** | Real boutique location on Contact page |
| **WhatsApp Web API** | `wa.me/923026825777` for all orders and forms |

## 🧩 Quick Customization Guide

| What | Where |
|---|---|
| Phone number | Search `923026825777` → replace in all `.html` files |
| Logo | Replace `images/logo.png` |
| Product images | Replace files in `images/` (keep same filenames) |
| Product names | `prefixNames` + `suffixNames` arrays in `collections.html` |
| Price ranges | `priceRanges` object in `collections.html` |
| Business hours | Footer of each page + `contact.html` |
| Brand colors | Tailwind config `<script>` block in `index.html` |
